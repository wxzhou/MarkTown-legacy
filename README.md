# MarkTown (Legacy Prototype)

This repository contains an early prototype of a WYSIWYG Markdown editor built from scratch.

⚠️ This project is no longer actively developed.

👉 A new version of MarkTown is being rebuilt in a separate repository: https://github.com/wxzhou/MarkTown

Status: legacy prototype, kept for reference.

- Developing tools: Trae + Claude Sonnet 3.7.
- Framework: node.js + electron (as suggested by ChatGPT). The developer knows neither of the two.
